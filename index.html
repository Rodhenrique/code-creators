Sprints

Nós usamos a metodologia scrum aonde divimos nosso projeto em 4 sprints

1 - Planejamento e design UIX

Planejamos as 4 sprints e fizemos levantamento de requisitos, separamos todas as tarefas. Começamos fazer o Design do website.

2 - Desenvolvimento do backend

Desenvolvemos o banco de dados em SQL Server e uma API em C# para conectar com SQL Server é devolver os dados requisitados.

3 - React Js Front-End

Começamos o desenvolvimento do Front-End do Website em React Js, usamos diversas bibliotecas do node. Tudo em TypesScrypt.

4 - React Native Mobile

Última sprint desenvolvimento do aplicativo mobile usado React Native, com diversas funcionalidades para facilita a vida da empresa e estudante do Senai.
